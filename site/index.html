Corpo

A calm, light-first, WCAG-AA design system for business software — the company-friendly sibling of cyberdesign. Plain CSS tokens and components, plus a full React API with Storybook docs.

What it is

  • 52 components, from Button to Spreadsheet
  • Five accent themes (teal default, amber, green, red, steel) plus a dark scope
  • Public Sans for body text, JetBrains Mono reserved for labels, data, and code
  • 2px/4px radii and quiet shadow elevation — no glow, no scanlines
  • MIT licensed, published as corpo (CSS) and react-corpo (React)
